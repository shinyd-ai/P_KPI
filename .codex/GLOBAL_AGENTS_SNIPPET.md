# Global AGENTS.md Snippet

Add the following to the global `AGENTS.md` when that file is available or when Codex has permission to write it.

```md
## Mandatory Implementation Planning

Before making any code, configuration, database, deployment, or filesystem change for a user-requested development task, first provide a detailed implementation plan.

The plan must include:

- Goal
- Scope
- Non-goals
- Files likely to change
- Implementation steps
- Verification steps
- Risks or assumptions

Do not start editing until the plan has been presented. For very small changes, the plan may be concise but must still be explicit. For broad work, create or update a project artifact such as `.codex/artifacts/design-plan.md`.
```
